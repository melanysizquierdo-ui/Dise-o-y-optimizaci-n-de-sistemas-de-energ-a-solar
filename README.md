# Dise-o-y-optimizaci-n-de-sistemas-de-energ-a-solar
Simulación educativa en Python para estimar la producción diaria de energía solar fotovoltaica. Calcula posición solar, irradiancia y potencia generada según ubicación, fecha y parámetros del panel. Incluye visualización gráfica y resultados energéticos. Ideal para estudiantes y técnicos.
